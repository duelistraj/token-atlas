# AGENTS

This repository uses the Token Atlas Project Knowledge Framework (PKF).

At the beginning of every session, read `.ai/PKF.md` before performing repository analysis.

## Code Search

- Prefer `rg` (ripgrep) for text search and file discovery.
- Prefer `sg` (ast-grep) for syntax-aware code searches and refactoring.
- Avoid `grep -r` unless `rg` is unavailable.

### IMPORTANT

This repo will not contain `.ai/PKF.md` as this repo belongs to the token-atlas skill itself. This repo is for maintaining and updating the skill. Do not run the token-atlas skill for this repo.
